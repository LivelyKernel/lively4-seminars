# Code Navigation
## The following functionalities were already supported:

| Shortcut     | Functionality           |
|--------------|-------------------------|
| Alt+Up       | Expand Selection        |
| Alt+Left     | Select Previous Element |
| Alt+Right    | Select Next Element     |
